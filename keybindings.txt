NEOVIM KEYBINDINGS REFERENCE
===========================

GENERAL/LEADER KEYS
------------------
Space           Leader key (<leader>)
\               Local leader key (<localleader>)

LSP & CODE NAVIGATION
-------------------
K               Hover Documentation
gd              Goto Definition
gD              Goto Declaration
gr              Find References
gi              Goto Implementation
gy              Goto Type Definition
<C-k>           Signature Help
<leader>rn      Incremental Rename
<leader>ca      Code Action
<leader>gf      Format
<leader>e       Show Line Diagnostics
[d              Next Diagnostic
[D              Previous Diagnostic

WHICH-KEY
---------
<leader>?       Show buffer local keymaps
<leader>!       Show all keymaps (including hidden)

TROUBLE
-------
<leader>xx      Toggle diagnostics
<leader>xX      Toggle buffer diagnostics
<leader>cs      Toggle symbols
<leader>cl      Toggle LSP (right window)
<leader>xL      Toggle location list

DEBUG
-----
<F5>            Debug actions
<leader>dt      Toggle breakpoint
<leader>dc      Continue debugging

FILE NAVIGATION & SEARCH
----------------------
<C-p>           Find Files
<leader>fg      Live Grep
<leader><leader> Recent Files
<C-n>           Explorer

TELESCOPE/FIND OPERATIONS
-----------------------
<leader>ff      Find Files
<leader>fg      Find with Grep
<leader>fb      Find Buffers
<leader>fr      Recent Files
<leader>fp      Projects

GIT OPERATIONS
------------
<leader>lg      LazyGit
<leader>gl      Git Log
<leader>gd      Git Definition
<leader>gr      Git References

SEARCH OPERATIONS
---------------
<leader>sb      Buffer Lines
<leader>sg      Grep
<leader>sw      Grep word under cursor
<leader>sd      Diagnostics
<leader>sD      Buffer Diagnostics
<leader>ss      LSP Symbols
<leader>sS      LSP Workspace Symbols

NOTIFICATIONS
------------
<leader>n       Notification History
<leader>un      Dismiss All Notifications

BUFFER OPERATIONS
---------------
<leader>bd      Delete Buffer
<leader>cR      Rename File

TOGGLE OPTIONS
------------
<leader>us      Toggle Spelling
<leader>uw      Toggle Wrap
<leader>uL      Toggle Relative Number
<leader>ud      Toggle Diagnostics
<leader>ul      Toggle Line Number
<leader>uc      Toggle Conceallevel
<leader>uT      Toggle Treesitter
<leader>ub      Toggle Dark/Light Background
<leader>uh      Toggle Inlay Hints
<leader>ug      Toggle Indent
<leader>uD      Toggle Dim

COMPLETION (nvim-cmp)
--------------------
<C-b>           Scroll docs backward
<C-f>           Scroll docs forward
<C-Space>       Complete
<C-e>           Abort completion
<CR>            Confirm completion

WORKSPACE
--------
<leader>wa      Add Workspace Folder
<leader>wr      Remove Workspace Folder
<leader>wl      List Workspace Folders

MISC
----
<C-/>           Toggle Terminal
]]              Next Reference
[[              Previous Reference
<leader>N       Neovim News
